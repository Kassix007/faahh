# Faahh

Faahh plays a sound when new errors appear in the VS Code Problems panel.

## Features

- Watches diagnostics for new errors.
- Plays `effect.wav` when the total error count increases.
- Provides a command to toggle the behavior on and off.

## Command

- `Faahh: Toggle On/Off`

## Setting

- `faahh.enabled`: Enable or disable sound playback.
